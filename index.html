# Welcome to Markdown

Markdown is a lightweight markup language that you can use to add formating elements to plaintext text documents

## How to use this ?

  • . Write markdown in the markdown editor window
  • . See the rendered markdown in the preview window

    • Features

  • - Create headings, paragraphs, links, blockquotes, inline-code, code-blocks and lists
  • - Name and save the document to access again later
  • - Choose between Light or Dark mood depending on your preference
